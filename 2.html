CSS

A Cascading Style Sheet, or Cascading Style Sheet, is a style language (English: Style sheet language[*]) that describes how a markup language is actually displayed[1], It is primarily used for XHTML, but can also be used for XML. It is a W3C standard and has a high degree of freedom when defining layouts and styles. The default file name [2] is style.css. If a markup language (ex: HTML) is in charge of the body of a website, CSS can be said to be in charge of decorating it like clothes and accessories. In other words, you can decorate a website like a completely different website by leaving the HTML structure as it is and changing only the CSS file. In the case of CSS3, which is currently under development, functions that can replace images created with graphic editing programs such as shadow effect, gradation, and transformation have been added. In addition, various animation functions have been added, replacing Adobe Flash to some extent.
